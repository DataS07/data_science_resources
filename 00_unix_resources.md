<font size="4em"><strong>Unix</strong></font>

#### Unix controls operations on the computer.

#### Unix Commands

###### Print working directory
```bash
pwd
```

###### Change directory to Home 
```bash
cd ~
cd 
cd $HOME
```

###### List files
```bash
ls
ls -l
ls -lha
ls -A
```
